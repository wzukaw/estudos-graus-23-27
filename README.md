# Página protegida para GitHub Pages

Publique esta pasta (`site_github_protegido`) no GitHub Pages.

Senha de acesso: não registre a senha no repositório público.

Observação: o conteúdo foi criptografado no arquivo JavaScript e só é inserido na página após a senha correta. Para proteção institucional robusta, use autenticação real em servidor; para GitHub Pages estático, esta é a opção mais adequada sem backend.
